#

>UCTF fetches upcoming CTFs from ctftime.org

### Prerequisites
- Python3.x

### Installation

- Clone the Repo
  ```
  git clone https://github.com/synacktraa/uctf
  ```

- Move to cloned directory
  ```
  cd uctf
  ```

- Now Run Tool
  ```
  python3 uctf.py --weight true --onsite false
  ```
